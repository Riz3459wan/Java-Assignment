# CSS Layout & Animation Project

## Project Structure

- index.html → Semantic HTML
- flexbox.html → Flexbox layouts
- grid.html → Grid layouts
- animations.html → Animations & transitions
- styles.css → Base styles
- flexbox.css → Flexbox styles
- grid.css → Grid styles
- animations.css → Animations styles
- assets/screenshots → Layout screenshots

---

## Features Implemented

### Layouts

- 12 Column Grid System
- Holy Grail Layout
- Dashboard Layout
- Magazine Layout
- Responsive Image Gallery

### Animations

- Button hover transitions
- Card flip animation
- CSS loading spinner
- Smooth scroll behavior
- Page fade-in transition

---

## Validation

All HTML files:

- Use proper DOCTYPE
- Include meta viewport
- Use semantic structure
- Pass W3C validation

---

## How to Run

Open any HTML file in a browser.
